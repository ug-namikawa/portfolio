# portfolio

## URL
https://ug-namikawa.github.io/portfolio/public/

## 使用方法

1. リポジトリをForkする

2. Forkしたリポジトリをローカルにダウンロードする

  ```Shell
  $ git clone URLを入れる
  ```

3. 設定する

  ```Shell
  // ダウンロードしたディレクトリに移動し、npm installする
  $ npm install
  ```

3. 動かす

  ```Shell
  $ npm start
  ```
